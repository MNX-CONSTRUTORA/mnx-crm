🔄 Sincronizar Dados
Como funciona a sincronização
Os dados ficam salvos em cada aparelho separadamente.
Para sincronizar, basta Exportar no aparelho de origem e Importar no destino.
Guarde o backup no iCloud Drive ou Google Drive para acessar de qualquer lugar.
📤 Passo 1 — Exportar (no aparelho principal)
Gera um arquivo .json com todos os seus leads e histórico.
Salve no iCloud Drive ou Google Drive.
📥 Passo 2 — Importar (no outro aparelho)
Abre o arquivo .json exportado e carrega os dados neste aparelho.
⚠️ Os dados atuais deste aparelho serão substituídos.
💡 Dica de fluxo diário:
1. Trabalhe no iPhone durante o dia
2. Ao final: Exportar → salvar no iCloud Drive
3. No iPad ou PC: Importar → selecionar o arquivo do iCloud
4. Dados sincronizados em todos os aparelhos ✅
📲 Instalar como App
📱 iPhone / iPad
  1. Abra o link do CRM no Safari
  2. Toque no ícone de Compartilhar (⬆️)
  3. Toque em "Adicionar à Tela de Início"
  4. Pronto — ícone MNX CRM na tela inicial! 🎉
💻 PC / Mac
  1. Abra o link do CRM no Chrome
  2. Clique nos 3 pontos → Salvar e compartilhar → Criar atalho
  3. Marque "Abrir como janela" → Criar
💡 Para sincronizar dados entre aparelhos, use o botão 🔄 Sincronizar na tela principal.
